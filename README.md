# BUNKR

**Smart attendance tracker for engineering students.** Calculate bunking scenarios instantly — no more paper + calculator math.

## Features

- 🤖 **AI Timetable Parsing** — Upload your college timetable image via Gemini Vision API
- 📊 **Real-time Calculations** — See projected attendance instantly as you toggle classes
- ⚠️ **75% Rule Guardian** — Never accidentally slip below the threshold
- 📅 **Day-wise Portal** — Swipe Mon–Fri to see each day's schedule
- ✅ **Select All / Bunk All** — One-click scenarios

## Tech Stack

- React + Vite
- Vanilla CSS (dark red-black theme)
- Google Gemini Vision API (free tier)

## Getting Started

1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Get a free [Gemini API Key](https://aistudio.google.com/app/apikey) and enter it on the setup screen

## Usage

1. Enter your current attendance numbers from your college portal
2. Upload a screenshot of your timetable
3. BUNKR parses it automatically (powered by Gemini AI)
4. On the dashboard, toggle any class to instantly see how your attendance changes
5. The 75% alert keeps you out of trouble
